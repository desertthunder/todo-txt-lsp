# Todo.txt Syntax Highlighting

This extensions provides syntax highlighting based on a TextMate grammar for todo.txt files.
In the future it will include embedded syntax highlighting for todo statements that
follow the [Todo.txt specification](#todotxt-specification).

## Overview

### Todo.txt Specification

Example: [x] (A) 2021-03-01 2021-02-28 +project @context key:value

Structure: status priority completion-date created-date description project context tags
